# BLOCO 1 – Fundamentos e Condições

## Exercício 1 – Sistema de Maioridade

Crie um programa que:

1.  Pergunte o nome do usuário
2.  Pergunte a idade
3.  Se a idade for maior ou igual a 18, mostre:  
    "Olá [nome], você é maior de idade."
4.  Caso contrário, mostre:  
    "Olá [nome], você é menor de idade."

---

## Exercício 2 – Sistema de Nota

Crie um programa que:

1.  Pergunte o nome do aluno
2.  Pergunte a nota (número decimal)
3.  Se a nota for maior ou igual a 10, mostre "Aprovado"
4.  Se a nota for maior ou igual a 8, mostre "Recuperação"
5.  Caso contrário, mostre "Reprovado"

Dica:  
Use `if`, `elif` e `else`.

---

## Desafio Extra

Crie um programa que:

1.  Peça um número ao usuário
2.  Informe se ele é:
    - Positivo
    - Negativo
    - Zero

## Exercício 3 – Contagem

## Exercício 4 – Cadastro de Nomes

Crie um programa que:

1.  Crie uma lista vazia
2.  Peça 3 nomes ao usuário
3.  Armazene os nomes na lista
4.  Mostre todos os nomes no final

---

## Exercício 5 – Soma de Números

Crie um programa que:

1.  Peça 5 números ao usuário
2.  Some todos os números
3.  Mostre o resultado final

Dica:  
Use uma variável acumuladora e `+=`.

# MINI PROJETO FINAL

## Projeto: Jogo de Adivinhação

Crie um programa que:

1.  Gere um número aleatório entre 1 e 10
2.  Peça ao usuário para adivinhar o número
3.  Informe se acertou ou errou

---

## Versão Melhorada (Desafio Final)

Melhore o jogo para:

1.  Continuar pedindo tentativa até o usuário acertar
2.  Contar quantas tentativas foram feitas
3.  Mostrar no final:

"Você acertou em X tentativas!"
